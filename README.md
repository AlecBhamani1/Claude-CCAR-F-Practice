# CCAR-F Exam Academy

An interactive, single-file study app for the **Claude Certified Architect – Foundation (CCAR-F)** exam.

🌐 **Live site:** https://alecbhamani1.github.io/Claude-CCAR-F-Practice/

## Features

- 3-week guided learning pathway
- 5 domain deep-dives
- 52 flashcards
- 72-question bank (Practice Tests A/B/C + 12 official sample questions) mapped to 34 exam objectives
- Domain drills
- **Score Report** tab — parses Pearson VUE score-report PDFs entirely in-browser and builds a ranked focus plan
- Guided practice-plan engine that adapts to your weakest domains
- Progress saved locally in your browser (`localStorage`)

Everything runs client-side — no server, no data leaves your browser.

## Development

The entire app is contained in [`index.html`](index.html). Open it in any modern browser to run locally.
